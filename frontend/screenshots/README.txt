Place screenshots in this folder. Name the Analytics screenshot `analytics-error.png`.

To capture the screenshot referenced in the README:
1. Open the app to the Analytics page in your browser. Reproduce the issue.
2. Open developer tools -> Console and take a screenshot showing the error and console messages.
3. Save the image as `analytics-error.png` and put it in this folder.

Once the file exists, it will render in the repository README as `frontend/screenshots/analytics-error.png`.
